
# Financial Risk Analysis Project

## Tools Used
- MS Excel
- SQL

## Project Description
This project analyzes loan default data to identify financial risk factors and categorize customers into Low, Medium, and High risk groups.

## Key Features
- Loan default dataset analysis
- Risk categorization using Excel formulas
- SQL-based data querying and aggregation
- Business insights for credit risk evaluation

## Files Included
- loan_default_data.xlsx : Dataset and Excel-based analysis
- risk_analysis.sql : SQL queries for risk analysis
